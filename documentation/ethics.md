# Riesgos eticos y limitaciones

- Los delitos reportados no equivalen a todos los delitos ocurridos.
- La vigilancia desigual puede sesgar los reportes.
- Un modelo predictivo no debe usarse para culpabilizar personas o barrios.
- Toda alerta debe presentarse con incertidumbre y revision humana.
- Las variables socioeconomicas pueden amplificar sesgos si no se auditan.
