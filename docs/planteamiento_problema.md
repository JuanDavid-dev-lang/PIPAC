# Planteamiento del Problema

PIPAC busca apoyar el analisis preventivo de criminalidad en Colombia mediante datos abiertos, limpieza automatizada, integracion territorial y modelos predictivos.

El problema central es transformar registros historicos dispersos en informacion accionable para identificar zonas, horarios y patrones de riesgo que ayuden a priorizar decisiones de seguridad ciudadana.

## Pregunta orientadora
