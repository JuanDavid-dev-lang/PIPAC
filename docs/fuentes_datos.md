# Fuentes de Datos

Las fuentes base provienen del portal oficial de datos abiertos de Colombia.

| Fuente | Dataset | Uso |
|---|---|---|
| datos.gov.co | Informacion delictiva del municipio de Bucaramanga | Registros historicos locales. |
| datos.gov.co | Reporte Hurto por Modalidades - Policia Nacional | Cobertura nacional de hurtos. |
| datos.gov.co | Accidentes de Transito ocurridos en el municipio | Proxy de movilidad local. |
| datos.gov.co | Proyeccion de poblacion de Bucaramanga | Calculo de tasas por territorio. |

Los archivos crudos se ubican en `data/01_raw/` y se mantiene una copia de compatibilidad en `datasets/raw/` para los modulos existentes.
