# Marco Metodologico

El proyecto sigue una adaptacion de CRISP-ML:

1. Comprension del problema y de los usuarios.
2. Inventario de fuentes abiertas y variables disponibles.
3. Ingesta desde datos.gov.co y archivos locales.
4. Limpieza, homologacion y enriquecimiento temporal.
5. Creacion de variables de riesgo y densidad territorial.
6. Entrenamiento de modelos supervisados y segmentacion espacial.
7. Evaluacion con metricas de clasificacion y revision cualitativa.
8. Publicacion mediante API, dashboard y frontend.
9. Documentacion de limitaciones, sesgos y uso responsable.
